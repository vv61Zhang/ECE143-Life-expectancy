Functions to display life-expectancy or other features on world map.

To install the relevant packages, run the following on Anaconda:
pip install pygal
pip install pygal_maps_world


In project_part1.py, we wrote the functions to display the graph required in the first part of the slides.
Top10 and bottom10 life expectancy data, as well as features.
